# community_questions — 社区开放问题（仅 Question）

从 **社交媒体、论坛、Discord 等** 收集的**尚未有可靠答案**的产品问题。  
每次收集或周检时更新 `last_checked`，并在帖子里找到官方/可靠回复后：

- 有完整 Q&A → 移到 `docs/inbox/` 或 `docs/faq/`
- 仅确认「仍无答案」→ 保持 `status: open`，更新 `last_checked`

## 客服 Agent 用法

- 可告知「社区有人提过类似问题，尚无官方结论」
- **不得**编造答案填补空白

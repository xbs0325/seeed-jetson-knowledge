* 问题
  * NVIDIA Jetson AGX Orin 64GB 官方开发套件有没有规格书、数据手册等资料？
* 适用产品
  * NVIDIA Jetson AGX Orin 64GB Developer Kit（官方套件，P3730）
* 适用平台类型：official_kit
* 简洁答案
  * **有。** 规格书、Developer Kit User Guide、模组 Data Sheet、Design Guide 等均在 **NVIDIA 官网** 与 **Jetson Download Center** 提供；部分文档需注册 NVIDIA 开发者账号下载。
* 注意事项
  * 先确认客户要的是 **NVIDIA 官方 Developer Kit**，不是 Seeed 载板/整机或其他第三方套件；后者资料路径不同。
  * 「Jetson AGX Orin 64GB」可能指 **64GB 内存的官方开发套件**，也可能被误说成仅模组；以 NVIDIA 官方 Developer Kit 文档为准。
  * 3D/CAD/机械图若在 NVIDIA 下载中心对应 Developer Kit 条目附件中，应引导客户到下载中心查看，未核对前不说「没有」。
  * 本仓库另有 [AGX Orin Developer Kit JetPack 7.2 更新路径](../official_kit/jetson-agx-orin-devkit/jetpack-7-2-iso-update.md)，偏刷机/BSP，不替代规格书。
* 关键规格（摘要，以 NVIDIA 官方文档为准）
  * AI 性能最高约 **275 TOPS**（64GB 版 Developer Kit）
  * **64GB LPDDR5**，**64GB eMMC**
  * 载板接口含 **16-lane MIPI CSI**、**10GbE**、PCIe x16（x8 Gen4）、M.2、USB 3.2、DisplayPort 等

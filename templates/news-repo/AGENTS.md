# Seeed Jetson 资讯 Agent

本仓库由 **Cursor Automation 每日定时**运行。

## 职责

记录 Jetson 相关**前沿技术、重大发布、版本更新**，写入 `daily/YYYY-MM-DD.md`。

## 来源

- NVIDIA Developer Blog / JetPack 发布页
- Seeed 新闻、Wiki 更新
- 重要 GitHub release（Linux_for_Tegra、jetson-containers 等）

## 规则

- 每条：标题、一句话摘要、链接、影响范围（官方 DevKit / Seeed 全系 / 某 SKU）
- 标注 `suggest_kb: yes|no`（是否建议写入技术支持库 1）
- 只开 **draft PR**，不 merge

完整指令：`instructions/news-digest.md`
